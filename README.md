# スタンプラリーシステム

## Pre Requisite

- NodeJS 18.0.0 over

## Setup

```bash
npm install
```

## Development Server

```bash
npm run dev
```

## Access

- stamp list
  `http://localhost:3000`
- get stamp
  `http://localhost:3000/stamps/1/collect`

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
